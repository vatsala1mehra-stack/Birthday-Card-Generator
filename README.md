# Birthday Card Generator

## Overview
The Birthday Card Generator is a simple web application that allows users to create customized birthday cards with personalized messages, images, and themes. Users can enjoy a quick and easy way to celebrate their loved ones' birthdays with heartfelt greetings.

## Features
- **Customizable Templates**: Choose from various pre-designed card templates.
- **Personalized Messages**: Add your own text to convey your unique wishes.
- **Image Uploading**: Upload images to include in your cards.
- **Sharing Options**: Share your cards via social media or download them as images.

## How to Use
1. Clone the repository:
   ```bash
   git clone https://github.com/vatsala1mehra-stack/Birthday-Card-Generator.git
   ```
2. Navigate to the project directory:
   ```bash
   cd Birthday-Card-Generator
   ```
3. Open the index.html file in your web browser.
4. Choose a template and fill in your message and image.
5. Share or download your card!
